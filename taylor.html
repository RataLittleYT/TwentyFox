API Documentation

Overview

Phatoms JWT Generator API provides programmatic access to JWT token generation for Garena Free Fire accounts. The API supports both single token generation and bulk processing capabilities.

Base URL: https://token-web.onrender.com/api/

Authentication

Currently, the API does not require authentication. However, rate limiting is enforced to prevent abuse.

Endpoints

POST /api/token
Description

Generate a JWT token for a single Garena account.

Request Body
{
  "uid": "1234567890",
  "password": "ABC123DEF456..."
}
Response
{
  "success": true,
  "data": {
    "uid": "1234567890",
    "status": "success",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "generated_at": "2024-01-01T12:00:00"
  }
}
POST /api/bulk-token
Description

Generate JWT tokens for multiple accounts by uploading a file.

Request

Send a multipart/form-data request with a file field.

Supported File Formats
  • JSON: Array of credential objects
  • Text: UID and password pairs
Response
{
  "success": true,
  "data": {
    "total_processed": 10,
    "successful": 8,
    "failed": 2,
    "results": [
      {
        "uid": "1234567890",
        "status": "success",
        "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "generated_at": "2024-01-01T12:00:00"
      },
      ...
    ]
  }
}
GET /api/download/<format>
Description

Download generated tokens in the specified format.

Parameters
  • format: Either 'json' or 'txt'
  • data: URL-encoded JSON data of tokens
Example
GET /api/download/json?data={"results":[...]}

Rate Limits

Endpoint Limit Window
